## 数据导入命令

```powershell
# 1. 删除旧数据库
Remove-Item -Force app_dev.db

# 2. 重建表结构
$env:PYTHONPATH = "项目根目录"
python -c "from app import create_app; from config import DevelopmentConfig; from app.extensions import db; app = create_app(DevelopmentConfig); app.app_context().push(); db.create_all(); print('OK')"

# 3. 导入基础数据（用户、食堂、精选菜品、吐槽、提报 + 9个基础档口）
python scripts/seed.py

# 4. 为所有食堂补充默认档口数据（匹配前端 mock 的 createDefaultCanteenStalls）
python scripts/seed_stalls_all.py
```
