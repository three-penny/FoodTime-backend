## 变更详情
1. Review 模型新增 status（默认 pending）和 audit_reason 字段，对齐 Rant 审核机制。
2. 管理员 API：
   - GET /reviews/admin：查询全部评价，批量关联 User/Dish/Stall/Canteen 表返回 reviewer_nickname + dish_name/stall_name/canteen_name。
   - PUT /reviews/:id/audit：通过或驳回评论。
3. 公开查询 get_reviews_by_dish 仅返回 status='approved' 的评论，确保未审核评论不可见。
4. dining_display_service._review_to_dict 新增 status 字段。
