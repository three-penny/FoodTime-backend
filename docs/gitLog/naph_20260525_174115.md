## 变更详情
1. 新增 `app/utils/auth_utils.py`，实现：
   - `generate_token()`：生成 JWT Token（HS256，含 user_id/account/role/exp）。
   - `decode_token()`：校验并解码 JWT Token。
   - `login_required` 装饰器：从 Authorization Header 提取 Bearer Token 并注入 g 对象。
   - `admin_required` 装饰器：基于 login_required 额外校验 role=admin。
   - `rate_limit` 装饰器工厂：基于 IP 的内存滑动窗口限流。
2. `auth_service.py` 增强：
   - `register()` 增加 `invite_code` 参数，role=admin 时校验邀请码。
   - 新增 `IntegrityError` 捕获与 rollback 处理。
   - 新增 `account` 重复注册检查。
   - `login()` 返回 JWT Token，增加 `account_status` 禁用检查。
3. `auth_routes.py`：登录接口添加 `@rate_limit(max_attempts=10, window_seconds=300)`。
4. `config.py`：新增 JWT 和邀请码相关配置项。
5. `requirements.txt`：新增 PyJWT 依赖。
